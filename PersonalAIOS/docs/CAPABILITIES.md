# PersonalAIOS: Complete Capabilities & Master Architecture Manifesto

## 1. Core Identity & Deployment
*   **Self-Hosted & Independent:** Not a SaaS. Runs locally on the user's physical laptop or private cloud instance. Complete data sovereignty.
*   **Dynamic Onboarding:** Users define the AI's name (e.g., Jarvis) and "Soul". By default, it acts as a "Savage Best Friend" (truthful, assigns side-quests for user improvement).
*   **Linguistic & Cultural Mastery:** Natively understands and replies in blended languages (like Hinglish) with perfect cultural nuance, slang, and context.

## 2. Omni-Channel Access (Remote & Local)
*   **Mobile Gateways:** Connected to WhatsApp, Telegram, and Discord. You can text your OS from your phone to execute complex tasks while away from your machine.
*   **Bi-directional Voice:** Real-time voice interaction capable of realistic human inflection, pausing, and interruption handling.
*   **Unconstrained UIs:** Offers a Terminal CLI for developers and a rich Web Dashboard for general users.

## 3. Physical Real-World Interaction (Eyes & Hands)
*   **Native Screen Vision:** Bypasses APIs by "looking" at screenshots using vision models to find UI coordinates, exactly like a human.
*   **Physical Actuation:** Takes those coordinates and uses the host OS (macOS/Win/Linux) to physically move the mouse and type organically, easily bypassing anti-bot protections.
*   **Webcam Sentiment:** Watches the user through the camera to read micro-expressions and emotional context (frustration, focus), dynamically altering its teaching style.

## 4. Unparalleled Autonomy & Capability
*   **Massive Long-Term Multitasking:** The EventBus and AsyncTaskQueue allow it to code a AAA game or complex physics engine in the background for months, while simultaneously responding to quick WhatsApp requests.
*   **Domain Spawner (PhD Level Learning):** If asked to do something it doesn't know (e.g., biochemistry), it autonomously spawns a Research Sub-Agent to scour the web, read academic papers, and learn it.
*   **Digital Surrogate (Passive Income/Creator):** Manages YouTube, Instagram, e-commerce, or algorithmic trading autonomously.
*   **Exploitative Problem Solver:** Unbound by strict "happy paths". It will find unconventional exploits and out-of-the-box methods to achieve verified goals.

## 5. Security & Immune System
*   **Skill Forge & Critic:** Automatically writes custom Python scripts for missing tools, but acts as its own critic to ensure the logic aligns with mathematical/physical axioms before saving it.
*   **Strict Sandboxing:** All generated code is run in fully isolated, disposable Docker containers to protect the host OS.
*   **Auto-Hacker (Red Team):** Actively pen-tests and hacks its own generated code to patch vulnerabilities (or finds bug bounties on the web).
*   **High-Risk Interceptor:** Protects against "Constructive Laziness". It halts and requests explicit confirmation before executing destructive commands (like formatting a drive) on the Host OS.

## 6. The Memory Domain
*   **Zero-VRAM Obsidian Graph:** Uses a local folder of heavily interconnected Markdown files. It pulls only the exact paragraph it needs using BM25/Regex, keeping the OS incredibly fast.
*   **The Sleep Cycle:** When the CPU is idle, a background process merges duplicate thoughts and archives stale data to maintain perfect structural organization.
